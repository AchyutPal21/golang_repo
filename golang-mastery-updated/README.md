# Go Mastery — Updated Curriculum

> Every module is broken into focused files, one topic per file.
> Each file is a **standalone runnable program** with deep explanations,
> internal mechanics, real-world context, and commentary on WHY things work the way they do.

## How to run any file

```bash
cd /home/achyut-pal/Desktop/upskill-go/golang-mastery-updated

go run 01_fundamentals/01_how_go_runs.go
go run 01_fundamentals/02_variables.go
# ... etc
```

## Study method per file

1. Read top to bottom — understand every comment block
2. Run it — observe the output
3. Modify values — predict what changes
4. Break something — read the compiler/runtime error, understand it
5. Write your own experiment at the bottom
